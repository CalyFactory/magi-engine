추천모듈 사용방법

---

추천 모듈 객체 생성!
```
from reco import Reco

recoModule = Reco(jsonData) 
```
첫번째 인자는 일정보강 모듈에서 받은 json데이터

추천 데이터 가져오기!
```
recoModule.getRecoList()
```
